# Repository Instructions (BSP)

## Language

- All content written to disk (docs/specs, code comments, HTML, markdown, etc.) must be in **English**.
- The user's instructions and the assistant's chat responses may be in Romanian.

## JavaScript Style

- Use Node.js built-ins only (no external runtime dependencies).
- Use ESM everywhere: `.mjs` files with `import`/`export` (no `require`/`module.exports`).
- Prefer `async/await` over Promise chains and callback-style Promise wrappers.
