Predictive Coding Theory

BSP is grounded in the Predictive Coding theory of cognition, popularized by neuroscientists like Karl Friston. This theory postulates that the brain is not a passive processor of inputs, but an active prediction machine.

Core Premise

The brain constantly generates top-down predictions about sensory input. It only processes (and learns from) the Prediction Error (or "Surprise"). If the input is perfectly predicted, no learning occurs.

From Brain to Software

In BSP, we model this explicitly:

  • Prediction: The system uses its current context (active groups) to guess what comes next.
  • Input: The actual text sequence arrives.
  • Surprise: We calculate the difference. Surprise = Input - Prediction.

This differs from standard Language Models (LLMs) which are trained to minimize cross-entropy loss over a massive dataset offline. BSP minimizes "Surprise" in real-time, online.
