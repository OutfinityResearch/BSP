The Learning Loop

Unlike Backpropagation which minimizes a global loss function across a fixed network using calculus (gradients), BSP performs local, incremental updates using set operations.

Hebb's Rule Revisited

Donald Hebb famously said: "Neurons that fire together, wire together."

BSP implements this literally:

  1. Group Creation: If features A, B, and C appear together often, they form a Group.
  2. Link Strengthening: If Group X is active at time T, and Group Y is active at time T+1, the link X->Y is strengthened.

Comparison with Backpropagation

Aspect Backpropagation (NN) BSP Learning
Scope Global (Updates all weights) Local (Updates active groups only)
Mechanism Chain Rule (Calculus) Set Union/Intersection
Speed Slow (Matrix Multiplications) Fast (Bitwise Ops)
Catastrophic Forgetting Major Issue Mitigated (Local interference only)

The Update Cycle

For every step in a chat session:

1. Calculate Surprise (Input \ Predicted)
2. If Surprise > Threshold:
   a. Update existing active groups (add new members)
   b. If no group fits, create a new Group from Surprise
3. Strengthen deduction links (Context -> Current)
4. Decay links that predicted incorrectly
